# Paisabazaar-Banking-Fraud-Analysis

The objective of this project is to analyze customer financial data to identify the key factors influencing credit scores and overall credit risk. Using Exploratory Data Analysis (EDA) and visualization techniques, the project aims to examine how variables such as income, outstanding debt, credit utilization, payment behavior, and loan history affect customer creditworthiness.

Additionally, the project seeks to identify unusual patterns and anomalies in financial behavior that may indicate potential risk or fraudulent activity. The insights derived from this analysis can help financial institutions improve credit risk assessment, enhance customer profiling, and make more informed lending decisions.
